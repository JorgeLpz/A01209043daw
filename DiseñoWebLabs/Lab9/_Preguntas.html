¿Cómo es que si el código está en un archivo con código html que se despliega del lado del cliente, se ejecuta del lado del servidor? Explica.

php es un lenguaje que se lleva acabo en el servidor, por lo tanto todo el script de php se encuentra dentro del servidro, todas las funciones llamadas sond entro del servidor, entonces cuando el cliente ejecuta un html estria llamando las funciones que se ejecutan dentro del servidor pero no en el navegador.
