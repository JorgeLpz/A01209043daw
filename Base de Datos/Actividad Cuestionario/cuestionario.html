¿Qué es entonces una base de datos - DB?

Es una colección de archivos interelacionados con la menor cantidad de repetición de elemntos entre ellos

¿En qué casos es conveniente usar bases de datos?

Cuando necesitas tener control de informacion y obtener consultas de dicha informacion que contiene muchos registros

¿Qué es un sistema de gestión de base de datos - DBMS?

Es una base de datos interelacionados que utliza programas para acceder a los datos contenidos dentro de la base de datos.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

Tener un programa para cada consulta necesaria.

  • Redundancia e inconsistencia de los datos
  • Dificultad para tener acceso a los datos
  • Aislamiento de los datos
  • Anomalías del acceso concurrente.
  • Problemas de seguridad
  • Problemas de integridad

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.
